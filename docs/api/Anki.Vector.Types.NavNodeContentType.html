    Enum NavNodeContentType

    The content types for a NavMapGridNode.

    Namespace: Anki.Vector.Types
    Assembly: Anki.Vector.dll
    Syntax
    public enum NavNodeContentType

    Fields

    Name Description
    NavNodeClearOfCliff

    The node is clear of any cliffs (a sharp drop) or obstacles.
    NavNodeClearOfObstacle

    The node is clear of obstacles, because Vector has seen objects on the other side, but it might contain a cliff. The node will be marked as either NavNodeCliff or NavNodeClearOfCliff once Vector has driven there.
    NavNodeCliff

    The node contains a cliff (a sharp drop).
    NavNodeInterestingEdge

    The node contains a visible edge (based on the camera feed).
    NavNodeNonInterestingEdge

    This entry is undocumented and not currently used
    NavNodeObstacleCube

    The node contains a LightCube.
    NavNodeObstacleProximity

    The node contains a proximity detected obstacle which has not been explored.
    NavNodeObstacleProximityExplored

    The node contains a proximity detected obstacle which has been explored.
    NavNodeObstacleUnrecognized

    The node contains an unrecognized obstacle.
    NavNodeUnknown

    The contents of the node is unknown.
